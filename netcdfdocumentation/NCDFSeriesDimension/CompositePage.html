NCDFSeriesDimension.h


Abstract

NCDFSeriesDimension objects for defining dimensionality of data variables.

Discussion

NCDFSeriesDimension is an immutable class designed to allow a programmer to access netcdf data over multiple files using standard notation. This class allows for dimensions to span files.

initWithDimension:

Initializing using a NCDFDimension exisiting in the root NCDFHandle. 

-(
        id)initWithDimension:(
        NCDFDimension *)aDim;
Parameter Descriptions
aDim
NCDFDimension object. Typically the root NCDFHandle from NCDFSeriesHandle.
Discussion

Initializes a new NCDFSeriesDimension. This initialization method is NOT for unlimited variables - limited variables only. This initializer will not span files.

initWithDimension: withHandleArray:

Initializing using a NCDFDimension exisiting in the root NCDFHandle. 

-(
        id)initWithDimension:(

        NSArray *)theHandles;
Parameter Descriptions
aDim
NCDFDimension object. Typically the root NCDFHandle from NCDFSeriesHandle.
theHandles
An NSArray of NCDFHandles owned by the NCDFSeriesHandle object.
Discussion

Initializes a new NCDFSeriesDimension. This method is for use when it is unknown whether the dimension is unlimited or limited. If unlimited, the dimension information will be adjusted to span multiple files.
