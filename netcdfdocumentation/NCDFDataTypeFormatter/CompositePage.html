NCDFDataTypeFormatter.h


Abstract

NCDFDataTypeFormatter is a subclass of NSFormatter designed to turn nc_types into NSString objects for display.

Discussion

This class is a convenience class for GUI interface elements, such as NSTableViews. Instead of requiring a conversion technique for each time the nc_type needs to be displayed, this formatter can be attached to a cell and the cell can be sent an NSNumber containing the nc_type value as an integer. It will automatically convert the number into a string: NC_BYTE, NC_CHAR, NC_SHORT, NC_INT, NC_FLOAT, and NC_DOUBLE.
