NCDFVariableByteSizeFormatter.h

NCDFVariableByteSizeFormatter is a subclass of NSFormatter designed to turn a number into a byte size report for display.


This class is a convenience class for GUI interface elements, such as NSTableViews. An integer value passed as an NSNumber object to this formatter with result in an NSString with the number converted into a byte size. It assumes that the number is the number of bytes, and converts that to b, kb, mb, or gbs depending on the value.
