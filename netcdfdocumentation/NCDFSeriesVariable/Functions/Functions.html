getAllDataInSlab

Returns a slab of data for later access using all data available for the variable.. 

-(
        NCDFSlab *)getAllDataInSlab;
Discussion

This method returns an NCDFSlab object containing the requested data. Accessing data from slabs allows faster data access than through file I/O and affords some thread safety. This method will span all files.

getValueArrayAtLocation:edgeLengths:

Returns selected data as an NSData object. 

-(
        NSData *)getValueArrayAtLocation:(
        NSArray *)startCoordinates edgeLengths:(
        NSArray *)edgeLengths;
Parameter Descriptions
startCoordinates
NSArray object with the coordinates of the data using NSNumber intValues. Values range from 0 to dimension length -1 for each dimension (in significance order).
edgeLengths
NSArray object with the lengths of the data along dimensions using NSNumber intValues. Values range from 1 to dimension length for each dimension (in significance order).
Discussion

Returns an NSData object containing all selected data. The data will automatically span files.

initWithVariable:fromHandle:

Initialize a new NCDFSeriesVariable using a NCDFVariable and NCDFSeriesHandle. 

-(
        id)initWithVariable:(
        NCDFVariable *)aVar fromHandle:(
        NCDFSeriesHandle *)aHandle;
Parameter Descriptions
aVar
NCDFVariable object. Typically the root NCDFHandle from NCDFSeriesHandle.
Discussion

Initializes a new NCDFSeriesVariable. This method should be considered private and should not be instanciated outside of an NCDFSeriesHandle. This variable class is also limited to variables that use an unlimited dimension, otherwise use a NCDFVariable object derived from a NCDFHandle.
