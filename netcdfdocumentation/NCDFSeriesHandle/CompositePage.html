NCDFSeriesHandle.h NCDFSeriesHandle provides an interface for using multiple netcdf files. If all the files have the same dimensions, data types, and represent a continuous sequence of data, this class provides an overall simple way to access the files without directly working with each. Also, you may store this class to disk.


Abstract

Handle for accessing multiple netcdf files that are sequential along the unlimited dimension.

Discussion

NCDFSeriesHandle provides an interface for using multiple netcdf files. If all the files have the same dimensions, data types, and represent a continuous sequence of data, this class provides an overall simple way to access the files without directly working with each. Also, you may store this class to disk.

getDimensions

Retrieves the dimensions of the receiver. 

-(
        NSArray *)getDimensions;
Discussion

The method provides an NSArray of NCDFSeriesDimension objects for all the known dimensions. Known dimensions only include those found in the root NCDFHandle object.

getRootDimensions

Provides the dimensions of the root object. 

-(
        NSArray *)getRootDimensions;
Discussion

The method provides the variables as an NSArray of NCDFDimension for the root handle.

getRootGlobalAttributes

Provides the global attributes from the root handle. 

-(
        NSArray *)getRootGlobalAttributes;
Discussion

The method provides the global attributes for the root handle. Global attributes can vary from file to file, but obtaining those attributes must be done through NCDFHandle objects.

getRootVariables

Provides the variables of the root object. 

-(
        NSArray *)getRootVariables;
Discussion

The method provides the variables as an NSArray of NCDFVariables for the root handle. This method is for accessing any variable from the root object.

getVariables

Retrieves the variables of the receiver. 

-(
        NSArray *)getVariables;
Discussion

The method provides an NSArray of NCDFSeriesVariable objects for all variables connected with the unlimited dimensions. All other variables are ignored.

initWithOrderedPathSeries:

Initialize a new NCDFSeriesHandle using an NSArray of netcdf file paths. 

-(
        id)initWithOrderedPathSeries:(
        NSArray *)paths;
Parameter Descriptions
paths
NSArray object containing paths to netcdf files.
Discussion

Initializes a new NCDFSeriesHandle object using an NSArray of file paths. The array must be ordered correctly since this initialization method does not attempt to correct the order based on the unlimited dimension data.

initWithOrderedURLSeries:

Initialize a new NCDFSeriesHandle using an NSArray of netcdf urls. 

-(
        id)initWithOrderedURLSeries:(
        NSArray *)urls;
Parameter Descriptions
paths
NSArray object containing urls to netcdf files.
Discussion

Initializes a new NCDFSeriesHandle object using an NSArray of urls. The array must be ordered correctly since this initialization method does not attempt to correct the order based on the unlimited dimension data. Note that urls should be file urls.

rootHandle

Provides the root NCDFHandle object 

-(
        NCDFHandle *)rootHandle;
Discussion

The root handle is an NCDFHandle object that provides static data that are not directly related to the unlimited dimension. Global attributes are an example. The root handle is typically the first handle in the receiver.
